# 右键长按助手

一个 macOS 小工具：点一下开始倍速，再点一下恢复。

默认推荐用“网页 3x”模式，它会直接把当前浏览器标签页里的视频设为 3 倍速。也可以切到“按住右方向键”模式，用系统级模拟按键控制支持该快捷键的软件。

## 使用

1. 双击打开 `右键长按助手.app`。
2. 打开 B 站或其他网页视频，让浏览器窗口保持在前台。
3. 点浮动面板里的“开始 3x”。
4. macOS 如果弹出“控制浏览器”的权限请求，选择允许。
5. 需要恢复时点“恢复 1x”。

## 模式

- `网页 3x`：推荐给 B 站、YouTube 等网页视频。支持 Chrome、Edge、Brave、Vivaldi、Opera、Arc、Safari。
- `按住右方向键`：给本地播放器或其他支持右方向键长按的软件使用。第一次使用需要在系统设置里允许辅助功能权限。

## 重新构建

```bash
./build.sh
```

构建完成后会生成：

```text
RightKeyHolder/右键长按助手.app
```

## 注意

- Safari 可能需要在 Safari 的开发菜单里允许“来自 Apple 事件的 JavaScript”。
- 如果当前光标在输入框里，右方向键会作用到输入框，所以使用按键模式前先点一下视频画面。
- 如果网页模式显示“请允许本工具控制浏览器”，到系统设置的“隐私与安全性 > 自动化”里允许它控制浏览器。
- 如果重新构建 app 后辅助功能权限看起来已经开启但仍然无效，退出 app，到“系统设置 > 隐私与安全性 > 辅助功能”里删除旧的 `右键长按助手`，再重新添加当前生成的 app。未上架签名的本地 app 每次重新构建后，macOS 可能会把它当成一个新的程序。
